langkah install project di lokal :
1. run "composer install"
2. buat database kosong dengan nama studiojingga
3. run "php spark migrate"
4. run "php spark db:seed ManageSeeder"
5. untuk menjalankan project, run "php spark serve"

langkah push ke git:
1. git add .
2. git commit -m "deskripsi_perubahan"
3. git push -u origin master
